# FUTURE_UX_02
# Glow Salon Appointment Booking App

## UI/UX Design Internship Task 2 – Future Interns

### Project Overview
This project is a mobile appointment booking application designed for Glow Salon.

The objective was to create a simple, user-friendly, and mobile-first booking experience that helps users schedule salon appointments efficiently.

---

## Business Chosen
Glow Salon

---

## Target Users
- Salon customers
- Working professionals
- Students
- Users seeking quick appointment booking

---

## Designed Screens
1. Splash Screen
2. Login / Signup Screen
3. Service Selection Screen
4. Specialist Selection Screen
5. Date & Time Slot Booking Screen
6. Appointment Confirmation Screen
7. Booking History Screen

---

## Key Features
- Mobile-first design
- Simple booking flow
- Service selection
- Specialist selection
- Date & time scheduling
- Appointment confirmation
- Booking history

---

## Tools Used
- Figma
- GitHub
- Google Fonts
- Coolors

---

## UX Decisions
- Minimal booking steps
- Large touch-friendly buttons
- Consistent color palette
- Clear visual hierarchy
- Easy navigation flow
- Reduced booking friction
